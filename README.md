# Road Accident Analysis Dashboard
## Project Overview
This project is an Excel-based Road Accident Analysis Dashboard created to analyze accident casualties and identify key patterns based on severity, vehicle type, road type, road surface, area, light condition, weather, and wind condition.
## Dashboard Features
- Total number of casualties
- Casualties by accident severity
- Casualties by vehicle type
- Current year vs previous year monthly trend
- Casualties by road type
- Casualties by road surface condition
- Casualties by area type
- Casualties by light condition
- Interactive filter panel for date, weather, and wind conditions
## Skills & Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Donut Charts
- Bar Charts
- Line Charts
- Dashboard Formatting
- Data Cleaning
## Purpose of the Project
The purpose of this dashboard is to convert road accident data into meaningful insights and help users understand accident trends, casualty patterns, and risk factors in a clear and visual way.
